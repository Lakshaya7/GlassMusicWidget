# 🎵 Glass Music Widget
A minimalist, transparent "punchhole" music widget for Windows. 

### ✨ Features
- **Minimalist Design**: 150x150 fixed size with rounded corners.
- **Punchhole Handle**: A transparent circle handle for dragging.
- **Smart Controls**: 
  - Single click album to Play/Pause.
  - Double-click left/right to skip tracks.
- **Windows Integration**: Works with Spotify, YouTube (via Chrome), and Media Player.

### 🚀 How to Run
1. Clone the repo:
   ```git clone https://github.com/YOUR_USERNAME/GlassMusicWidget.git```
2. Install dependencies
   ```pip install -r requirements.txt```
3. Run:
   ```python src/music_widget.py```
   (Run as Administrator for Media access).

### 🛠️ Built With
- Python 3.x
- PyQt6 (UI)
- Winsdk (Windows Media Session)
